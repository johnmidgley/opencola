Welcome to OpenCola!

In order to access OpenCola over a secure connection, you need to install an SSL certificate. To start the process, click Download Certificate and then follow the instructions below.

Instructions

After the certificate is downloaded, open it:

Click Install Certificate:

Click Next:

Select Place all certificates in the following store:

Click Browse

Select Trusted Root Certificate Authorities and click OK and then Next:

Click Finish, which will likely display a security warning:

Click Yes, then click Yes on the confirmation dialog and close the wizard.
