# Install extension

- In a Chrome based browser (maybe works on Firefox?), got to extensions (e.g. brave://extensions/)
- Select "Developer Mode"
- Click "Load unpacked"
- Select the 'chrome' directory in this folder (opencola)
- Pin the extension to the toolbar
